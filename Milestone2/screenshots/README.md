# Milestone 2 – Policy Navigator

This project is part of the Infosys Springboard Internship – Milestone 2.

## Features Implemented
- OTP based user authentication
- Secure login flow using Streamlit session state
- Readability Dashboard for policy analysis
- Improved UI and user experience

## Application Flow
1. User enters email ID
2. OTP is generated and verified
3. After successful authentication, user is redirected to dashboard
4. User can analyze policy text for readability metrics
